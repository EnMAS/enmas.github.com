org.enmas.pomdp

State

class State extends Serializable

Represents the state of a POMDP.

This structure is a type-safe wrapper for a totally generic HashMap. The State stores type information for mapped objects. When retrieving objects, the proper type must be supplied.

Note: This class should uses ClassTags instead of TypeTags due to a bug in Scala 2.10 where TypeCreators are not serializable.
